## 第一步：装 Python（已有可跳过）

> ⏱ 预计 3 分钟

1. 打开 [python.org](https://www.python.org/downloads/) → 点黄色按钮下载
2. 打开下载的文件 → **⚠️ 勾选「Add Python to PATH」** → Install Now

**验证：** `Win+R` → 输入 `cmd` → `python --version` → 显示版本号即成功

---

## 第二步：装 ZSXQ Fetcher

> ⏱ 预计 2 分钟

1. 解压 `ZSXQ-Fetcher.zip` 到任意文件夹
2. 双击 `setup.bat` → 看到「安装完成」

> ❓ 一闪而过？在文件夹地址栏输入 `cmd` 回车，手动输入 `setup.bat` 看报错

---

## 第三步：获取 Token（🔑 最重要的一步）

> ⏱ 预计 2 分钟 | ⚠️ Token 7-30 天过期需重取

### 3.1 登录网页版

浏览器打开 [wx.zsxq.com](https://wx.zsxq.com) → 微信扫码登录

### 3.2 提取 Token

1. 按 `F12`（或右键 → 检查）
2. 顶部点 **Application**（应用程序）→ 没看到就点 `>>`
3. 左侧 **Cookies** → `wx.zsxq.com`
4. 右侧找到 `zsxq_access_token` → **双击值 → Ctrl+C 复制**

### 3.3 写入配置

打开 `scripts\config.json`，把 token 粘贴进去：

```json
{ "zsxq_access_token": "你的token粘贴在这里" }
```

## 第四步：日常使用

```bash
# 列出所有星球
python scripts/zsxq_fetcher.py --list

# 下载帖子（默认 20 条）
python scripts/zsxq_fetcher.py

# 下载指定数量
python scripts/zsxq_fetcher.py --count 50

# 关键词搜索
python scripts/zsxq_fetcher.py --search "副业"

# 导出 Markdown
python scripts/zsxq_fetcher.py --format markdown
```

下载内容在 `data/` 文件夹：帖子存为 `.md` 文件，附件在 `files/` 目录。

## 第五步：接入 AI（强烈推荐 ⭐）

接入后你可以直接用自然语言操作："帮我总结最近一周的精华帖"、"搜一下AI相关的讨论"

### 支持的工具

| 工具 | 接入方式 | 一句话说明 |
|------|----------|-----------|
| **Claude Desktop** | MCP 配置 | 最成熟的 MCP 生态 |
| **Codex（桌面版）** | MCP 配置 | OpenAI 官方桌面 AI |
| **Cursor** | MCP 配置 | 写代码时顺便读星球 |
| **Windsurf** | MCP 配置 | 同上，AI IDE |
| **VS Code + Continue** | MCP 配置 | 同上 |
| 通义千问 / Kimi / DeepSeek / 豆包 | 手动粘贴 | 先下载 md，复制给 AI |

### 通用 MCP 配置（Claude / Codex / Cursor / Windsurf 均适用）

编辑对应工具的 MCP 配置文件，加入：

```json
{
  "mcpServers": {
    "zsxq": {
      "command": "python",
      "args": ["你的安装路径\\scripts\\zsxq_mcp_server.py"],
      "env": { "ZSXQ_TOKEN": "你的token" }
    }
  }
}
```

| 工具 | 配置文件位置 |
|------|-------------|
| Claude Desktop | `%APPDATA%\Claude\claude_desktop_config.json` |
| Codex | 设置 → MCP Servers → 添加 |
| Cursor | 项目根目录 `.cursor\mcp.json` |
| Windsurf | 设置 → MCP Servers → 添加 |

> ⚠️ 配置后**完全关闭**工具再重新打开

### MCP 能干什么？

| 你对 AI 说 | AI 自动做的事 |
|-----------|-------------|
| "列出我的星球" | 读取星球列表和帖子数 |
| "搜一下关于AI的帖子" | 关键词全文搜索 |
| "打开第3条看看" | 显示帖子详情 |
| "总结前10条" | 批量阅读+提炼要点 |
| "这篇里的PDF能下载吗" | 获取附件下载链接 |

### 国产 AI 怎么用？

国产工具暂不支持 MCP 自动读取，但可以手动操作：

```bash
# 1. 下载并导出
python scripts/zsxq_fetcher.py --format markdown --count 20

# 2. 打开 data/ 下生成的 .md 文件

# 3. 全选复制 → 粘贴到 AI 对话框 → 说：
#    "帮我总结这些帖子的核心观点"
```

## 常见问题

| 问题 | 答案 |
|------|------|
| 双击 setup.bat 一闪而过 | 文件夹地址栏输 `cmd` 回车，手动打 `setup.bat` |
| 「python 不是内部命令」 | 重装 Python，**必须勾选 Add to PATH** |
| Chromium 下载失败 | 网络问题，重试或开代理 |
| 提示 1059 / 签名错误 | Token 过期，回第三步重新获取 |
| 下载慢？ | 正常，模拟浏览器防反爬，20条约1-2分钟 |
| 手机能用吗？ | 需要电脑。手机只能看网页版星球 |
| Mac 能用吗？ | ✅ 可以，装 Python 后在终端执行同样命令 |
| 报错怎么办？ | 90% 是 token 过期 → 重取即可。还不行就截图发群 |
